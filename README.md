# 🌸 Bloom

## 📌 프로젝트 소개
**Bloom**은 Flask 기반으로 제작한 **웹 애플리케이션**으로,  
데이터 입력 및 조회 기능을 중심으로  
**백엔드와 프론트엔드 연동 흐름**을 학습하기 위해 제작한 프로젝트입니다.

Python Flask를 사용하여 서버를 구성하고,  
HTML, CSS, JavaScript를 활용해  
웹 애플리케이션의 기본 구조와 동작 원리를 이해하는 것을 목표로 합니다.

---

## 🛠 사용 기술
- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite (로컬 개발용)
- **IDE**: VS Code
- **Version Control**: Git, GitHub

---


## ✨ 주요 기능
- 웹 폼을 통한 데이터 입력
- 데이터베이스 연동 및 데이터 저장
- 저장된 데이터 조회 기능
- Flask 라우팅을 통한 페이지 처리
- 프론트엔드 ↔ 백엔드 데이터 흐름 구현

---

## 🧠 프로젝트를 통해 배운 점

- Flask 기반 웹 서버 구성 경험
- 템플릿 엔진(Jinja2)을 활용한 화면 구성
- static / templates 디렉터리 구조 이해
- 웹 애플리케이션 기본 CRUD 흐름 이해
- GitHub를 활용한 프로젝트 버전 관리 경험
